Instance Method

openSearchInNavController(_:animated:centerID:uneco:brand:infoPlace:)

Present in the UINavigationController a new map of the centre visited by user with the place seasrched marked and its info open

Declaration

func openSearchInNavController(_ navController: UINavigationController, animated: Bool, centerID: Int, uneco: String, brand: String, infoPlace: infoplace?)

Discussion

  • navController: UINavigationController where the view controller will be pushed

  • animated: Specify true to animate the transition or false if you do not want the transition to be animated.

  • center: Identifier of the center

  • uneco: the uneco code to search the place

  • brand: brand to search the place

  • infoPlace: info to use in a custom view, if this is nil the place info will be use
