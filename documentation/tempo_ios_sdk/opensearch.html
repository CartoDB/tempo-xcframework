Open indoor map sdk from search on App

Open the Indoor Map SDK, with a space selected where the search element can be find, and the search view.

Overview

When an user search a product on the App, if he is in a center or maybe if he has selected a preferred center. It must be possible to show the search product on the center map.

The App should use the next function in orther to communicate with SDK

func openSearchInNavController(_ navController: UINavigationController,
                                           animated: Bool,
                                           centerID: Int,
                                           uneco: String,
                                           brand: String,
                                           infoPlace: InfoPlace?)

Information to find the properly space.

The app should send to SDK the uneco info and the brand in other to find the space.

Info to show the user

To show the user on the map the element searched, at the begging and when route is completed, the App should send the InfoPlace struct.
