Instance Method

getCentersWithIndoorEnabled(completion:)

Use this function to obtain the list of centers that have the indoor functionality.

Declaration

func getCentersWithIndoorEnabled(completion: @escaping (Result<[Int : indoorcenter], Error>) -> Void)

Parameters

completion

A closure executed in the main thread when the centers fetch is completed. When successful, its parameter contains all the centers that have the indoor functionality enabled. For convenience, this parameter is a dictionary where the keys are the ids of the centers and the values are the IndoorCenter objects.
